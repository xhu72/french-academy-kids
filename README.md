# French Academy App

French Academy Kids is a web-based French language learning platform designed for learners of all ages, from Grade 1 through Grade 12. The app makes learning French fun and accessible through bright, playful visuals, interactive games, and a structured progression system that mirrors school grade levels.

Users sign up with an email and password, then progress through four locked difficulty levels. Each level covers a specific set of vocabulary topics and introduces new question types as complexity increases. Points are earned by passing quizzes, and the next level unlocks automatically when the pass mark is achieved.
